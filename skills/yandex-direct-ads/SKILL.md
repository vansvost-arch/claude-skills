## Обзор

Скилл для автоматического формирования рекламных кампаний в Яндекс Директ.
Работает с товарами TDCompass — промышленная химия (клеи, герметики, смазки).

## Стратегия TDCompass

**Принцип:** Берём за счёт покрытия, а не за счёт дорогих позиций.

- НЧ-ключи (20-100 запросов/мес): основа, дешёвые клики 5-15₽
- СЧ-ключи (100-1000): умеренная конкуренция, 15-30₽
- ВЧ-ключи (>1000): осторожно, высокая конкуренция, ручное управление
- Микро-НЧ (<20): не рекламируем, только SEO

## Группировка кампаний

Формат имени: `{Категория}_{Частотность}_{Тип}`

Примеры:
- `Герметики_НЧ_Поиск`
- `Герметики_СЧ_Поиск`
- `Герметики_НЧ_РСЯ`
- `Клеи_НЧ_Поиск`

Правила:
1. Поиск и РСЯ — всегда отдельные кампании
2. НЧ, СЧ, ВЧ — отдельные кампании (разные ставки)
3. По категориям (герметики, клеи, смазки, и т.д.)

## Воркфлоу создания кампании

### Шаг 1: Анализ товара
```
Входные данные:
- Product из БД (name, description, category, price, images)
- DemandSnapshot (ключевые слова с частотностями)
- CompetitorAd (объявления конкурентов)
```

### Шаг 2: Определение кампании
```
1. Определить категорию товара
2. Для каждого ключевого слова определить частотность (ВЧ/СЧ/НЧ)
3. Определить кампанию: {Категория}_{Частотность}_{Поиск|РСЯ}
4. Если кампания существует — добавить группу, если нет — создать
```

### Шаг 3: Создание группы объявлений
```
Группа = один товар внутри кампании
- Название: "{Бренд} {Модель}"
- Минус-слова: конкуренты, нерелевантные добавки
```

### Шаг 4: Создание объявлений

#### Для Поиска:
```
Заголовок 1 (до 56 симв.): Главный ключ + бренд
  Пример: "Sika 268 Pro — полиуретановый герметик"

Заголовок 2 (до 30 симв.): УТП или CTA
  Пример: "В наличии. Доставка по РФ"

Текст (до 81 симв.): Преимущества + CTA
  Пример: "Профессиональный герметик для вклейки стёкол. Сертификат. Цена от 1200₽. Закажите!"

Отображаемая ссылка (до 20 симв.):
  Пример: "герметики/sika"

Быстрые ссылки (до 8 шт × 30 симв.):
  1. "Характеристики" → /catalog/sika-268-pro#specs
  2. "Сертификаты" → /catalog/sika-268-pro#docs
  3. "Аналоги" → /catalog/germetiki
  4. "Доставка и оплата" → /delivery

Уточнения (до 4 шт × 25 симв.):
  1. "Бесплатная доставка"
  2. "Оригинал с сертификатом"
  3. "Оптовые цены"
  4. "Консультация эксперта"
```

#### Для РСЯ:
```
Заголовок 1: Более эмоциональный, привлекающий внимание
  Пример: "Ищете надёжный герметик для стёкол?"

Заголовок 2: Конкретное предложение
  Пример: "Sika 268 Pro от 1200₽"

Текст: Развёрнутое описание + визуал
  Пример: "Профессиональный полиуретановый герметик Sika. Используют автосервисы по всей России."

Изображение: Фото товара (если есть в images[])
```

### Шаг 5: Ключевые слова
```
Типы соответствия:
- Фразовое (по умолчанию): "sika 268 pro" — показ по запросам содержащим фразу
- Точное: "!sika !268 !pro" — только точная форма слов
- Широкое: sika 268 pro — все формы и синонимы

Для НЧ: фразовое соответствие (максимальный охват)
Для СЧ: фразовое + минус-слова (фильтрация нецелевых)
Для ВЧ: точное соответствие (контроль расхода)

Минус-слова:
- Общие: "бесплатно", "своими руками", "отзывы", "форум", "реферат"
- Категорийные: для герметиков — "силиконовый" (если товар полиуретановый)
- Конкуренты: "leroy", "obi", "castorama" (если не хотим показываться)
```

### Шаг 6: Стратегии назначения ставок
```
Поиск НЧ:
  Стратегия: "Максимум кликов"
  Ограничение: средняя CPC ≤ 15₽
  Дневной бюджет: 300₽

Поиск СЧ:
  Стратегия: "Максимум кликов"
  Ограничение: средняя CPC ≤ 25₽
  Дневной бюджет: 500₽

Поиск ВЧ:
  Стратегия: Ручное управление
  Начальная ставка: по прогнозу Директа
  Дневной бюджет: 1000₽

РСЯ:
  Стратегия: "Максимум кликов"
  Ограничение: недельный бюджет
  CPC обычно дешевле поиска в 2-3 раза
```

### Шаг 7: Расширения и дополнения
```
Виртуальная визитка:
  - Название: TDCompass
  - Телефон: (из .env)
  - Адрес: (если есть)
  - Время работы: Пн-Пт 9:00-18:00

Счётчик Метрики:
  - Привязать счётчик для отслеживания конверсий
  - UTM-метки: utm_source=yandex&utm_medium=cpc&utm_campaign={campaign_name}

Отслеживание:
  - Разметка ссылок (yclid)
  - Электронная коммерция в Метрике
```

## API Яндекс Директ v5

Base URL: `https://api.direct.yandex.com/json/v5/`

### Создание кампании
```bash
curl -X POST "https://api.direct.yandex.com/json/v5/campaigns" \
  -H "Authorization: Bearer $YANDEX_DIRECT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "method": "add",
    "params": {
      "Campaigns": [{
        "Name": "Герметики_НЧ_Поиск",
        "StartDate": "2026-03-10",
        "TextCampaign": {
          "BiddingStrategy": {
            "Search": {
              "BiddingStrategyType": "WB_MAXIMUM_CLICKS",
              "WbMaximumClicks": {
                "WeeklySpendLimit": 5000000000,
                "BidCeiling": 15000000
              }
            },
            "Network": {
              "BiddingStrategyType": "SERVING_OFF"
            }
          },
          "Settings": [
            {"Option": "ADD_METRICA_TAG", "Value": "YES"},
            {"Option": "ADD_OPENSTAT_TAG", "Value": "NO"},
            {"Option": "ADD_TO_FAVORITES", "Value": "NO"},
            {"Option": "ENABLE_AREA_OF_INTEREST_TARGETING", "Value": "YES"},
            {"Option": "ENABLE_COMPANY_INFO", "Value": "YES"},
            {"Option": "ENABLE_SITE_MONITORING", "Value": "YES"},
            {"Option": "REQUIRE_SERVING_BY_PLACEMENT_TYPE", "Value": "NO"}
          ]
        }
      }]
    }
  }'
```

### Создание группы объявлений
```bash
curl -X POST "https://api.direct.yandex.com/json/v5/adgroups" \
  -H "Authorization: Bearer $YANDEX_DIRECT_TOKEN" \
  -d '{
    "method": "add",
    "params": {
      "AdGroups": [{
        "Name": "Sika 268 Pro",
        "CampaignId": 12345,
        "RegionIds": [225]
      }]
    }
  }'
```

### Создание объявления
```bash
curl -X POST "https://api.direct.yandex.com/json/v5/ads" \
  -H "Authorization: Bearer $YANDEX_DIRECT_TOKEN" \
  -d '{
    "method": "add",
    "params": {
      "Ads": [{
        "AdGroupId": 67890,
        "TextAd": {
          "Title": "Sika 268 Pro — герметик полиуретановый",
          "Title2": "В наличии. Доставка по РФ",
          "Text": "Для вклейки автостёкол. Сертификат. Цена от 1200₽. Закажите!",
          "Href": "https://tdcompass.ru/catalog/sika-268-pro?utm_source=yandex&utm_medium=cpc",
          "DisplayUrlPath": "герметики/sika",
          "SitelinkSetId": 111,
          "AdExtensionIds": [222, 333]
        }
      }]
    }
  }'
```

### Добавление ключевых слов
```bash
curl -X POST "https://api.direct.yandex.com/json/v5/keywords" \
  -H "Authorization: Bearer $YANDEX_DIRECT_TOKEN" \
  -d '{
    "method": "add",
    "params": {
      "Keywords": [
        {"AdGroupId": 67890, "Keyword": "sika 268 pro купить", "Bid": 15000000},
        {"AdGroupId": 67890, "Keyword": "герметик sika 268 цена", "Bid": 12000000}
      ]
    }
  }'
```
